# 資格・リスキリング講座ナビ

## プロジェクト概要

教育訓練給付金の対象講座（約15,000件）を検索・比較できるSEOメディア。
中立の立場で各スクールを比較し、「この資格、補助金使えば実質◯万円」がわかるサイト。

## 技術スタック

- **言語**: Python 3.12
- **テンプレートエンジン**: Jinja2
- **DB**: SQLite
- **CI/CD**: GitHub Actions
- **ホスティング**: Cloudflare Pages

## フォルダ構成

```
data/           # データ取得スクリプト・SQLite DB
generators/     # 静的サイト生成スクリプト
templates/      # Jinja2 HTMLテンプレート
articles/       # マークダウン記事ソース
static/         # CSS・画像などの静的ファイル
output/         # 生成済みHTML（Cloudflare Pagesのデプロイ対象）
```

## SEO記事ルール

- **タイトル**: 30〜60文字
- **H2**: 1記事あたり3〜6個
- **meta description**: 80〜120文字
- **内部リンク**: 最低3本／記事
- **文体**: です・ます調

## アフィリエイト単価目安

| カテゴリ | 単価 |
|---|---|
| プログラミングスクール | 1〜2万円／件 |
| 英会話スクール | 5,000〜15,000円／件 |
| 資格スクール | 3,000〜8,000円／件 |

## 開発ルール

- `output/` 以下のファイルは自動生成のため手動編集禁止
- データ更新は `data/` 配下のスクリプトで行う
- テンプレート変更後は必ず `generators/build.py` を実行して動作確認する
- コミット前に生成済みHTMLのバリデーションを確認する
